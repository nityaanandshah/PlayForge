# PlayForge Frontend

React + TypeScript frontend for PlayForge multiplayer gaming platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Start development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router v6
- Axios
- Zustand (state management)
- TailwindCSS
